# shiny kuzco app

a simple wrapper of kuzco to make computer vision for everyone. few-shot
via frank hull and shiny assistant
(https://gallery.shinyapps.io/assistant/)

## Usage

``` r
kuzco_app()
```

## Value

a shiny app instance as a playground for local llms

## Examples

``` r
if (FALSE) { # \dontrun{
kuzco_app()
} # }
```
