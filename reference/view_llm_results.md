# view llm results as a tidy great table

view llm results as a tidy great table

## Usage

``` r
view_llm_results(llm_results)
```

## Arguments

- llm_results:

  results from one of the llm_image\_\* functions

## Value

a great table to view the results neatly

## Examples

``` r
if (FALSE) { # \dontrun{
view_llm_results(llm_image_alt_text())
} # }
```
